装机常用软件合集

这里记录一些windows系统装机后自用的一些常用软件

浏览器

Microsoft Edge

虽然现在windows11系统的电脑自带新版Edge浏览器，但是由于固有印象的原因使用的人可能并不多，我还是要说一下Edge浏览器发展到今天已经足够好用，只需登录微软账户即可在PC、手机、平板多端同步历史记录/书签/密码等数据，非常省心。

Firefox

第三方浏览器我一直使用火狐浏览器，功能和Edge大同小异，主要是里面的插件用习惯了。值得一提的是火狐有中国版和国际版两个版本，云端数据不互通。这里链接的是中国版，数据同步方面体验较好。

生产力软件

有道云笔记

国产软件多端同步比较省心，用来放一些临时写的笔记。

Typora

老牌 Windows / macOS 平台 Markdown 写作工具,正版89RMB。用来在电脑上编辑markdown文档，写作博客时用的比较多。

SM.MS

编写markdown文档比较头疼的是处理图片内容。不同于word等文档编辑器，图片内容直接储存在文件中，markdown作为一个纯文本编辑器，不会直接储存图片数据，而是需要将图片先上传到一个第三方图片储存网站，即所谓的“图床”，再记录下图片的图床链接，最终markdown文档中只含有图片的图床链接。SMMS即我使用的图床，个人免费使用有5GB的储存空间。

PicGo

PicGo是一个用于快速上传图片并获取图片 URL 链接的工具，配合图床使用。

Foxmail

自用邮件客户端软件，主要是工作需要使用。

To Do

Microsoft To Do是微软家的待办清单，多端同步体验很好，对于轻量使用者很方便。

可以直接在微软商店搜索安装。

实用工具

PowerToys

Microsoft PowerToys 是微软官方发布的一款实用软件，可帮助高级用户调整和简化其 Windows 体验，从而提高工作效率。其集成了许多效率工具，诸如窗口管理器、文本提取、快速搜索等，功能十分丰富。

可以直接在微软商店搜索安装。

QuickLook

一款便捷的文件预览软件，选中一个文件，按下空格，就会快速弹出一个窗口来预览文件内容，平时对于文本、照片、视频、压缩包、pdf等等文件都可以空格预览看看，非常的方便，甚至还可以在作者的Github项目下载更多的插件来支持更多格式文件的预览。

可以直接在微软商店搜索安装。

TranslucentTB

一款美化Windows任务栏的小软件，可以让任务栏变得全透明，当然也支持其他样式，比如磨砂、半透明等效果。

可以直接在微软商店搜索安装。

娱乐项目

Steam

无需多言，就是百度首页全是盗版网站，需要注意甄别。
